Java基础

Java三个主要版本

  • Java SE：标准版，用于桌面应用程序开发，也是Java的基础
  • Java EE：企业版，用于开发企业级分布式的网络程序
  • Java ME：用于嵌入式系统开发，需要底层操作系统支持

特性

  • 简单的（simple）
  • 面向对象的（object-oriented）
  • 分布式的（distributed）
  • 解释性的（interpreted）
  • 健壮的（robust）
  • 安全的（secure）
  • 结构中立的（architecture neutral）
  • 便捷的（portable）
  • 高性能的（high-performance）
  • 多线程的（multithreaded）
  • 动态的（dynamic）

面向对象编程

  • 面向过程（Procedure Oriented Programming）简称 POP
    • 基础的顺序思维，过程为中心
    • 核心问题是数据结构和算法的开发和优化
  • 面向对象（Object Oriented Programming）简称 OOP
    • 按客观世界思维方式，以对象为中心
    • 直接重现问题中各个对象之间的关系
    • 基本特性：抽象、继承、封装、多态

标识符、注释

  • 标识符由数字、字母、下划线、美元符组成
  • 严格区分大小写

注释

  • 单行注释：//开头

  • 多行注释：“ /* “开头，” */ “结尾

  • 文档注释：“ /** ” 开头，“ */ ”结尾

常量、变量

常量

  • 使用前必须定义，需要初始化

  • 初始化后不可修改

  • 使用final关键字定义

  • 常用大写字母表示，区分变量

  • 静态常量定义

常量值

  • 整型：十进制、八进制（0开头）、十六进制（0X或0x开头）

  • 长整型：整型后面以 L 结尾

  • 浮点型：十进制由数字和小数点组成，科学记数法其中e或E之前必须有数字，之后的数字必须为整数

  • 布尔型：true或false

  • 字符型：单引号括起来的单个字符（数据类型为char）以及转义字符（以 \ 开头的字符序列）

  • 字符串：双引号括起来的若干个字符
