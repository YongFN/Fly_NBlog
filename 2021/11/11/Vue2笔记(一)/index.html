创建vue实例

总结：

  1. 要让Vue工作，必须创建一个Vue实例，并传入一个配置对象
  2. root容器里的代码依然符合html规范，只是混入了一些特殊的vue语法
  3. root容器里的代码称为Vue模板
  4. Vue实例和容器是一一对应的
  5. 真实开发中只有一个Vue实例，并且配合组件一起使用
  6. {{xxx}}中的xxx要写js表达式，xxx可以自动读取到data中的所有属性
  7. data中的数据发生改变，页面中用到该数据的地方会自动更新

补充：js表达式和js代码（语句）的区分

表达式：一个表达式会产生一个值，可以放在任何一个需要值的地方

  • a
  • a + b
  • demo(1)
  • x === y ? 'a' : 'b'

js代码（语句）：

  • if(){}
  • for(){}

Vue中还有很多的指令，形式都是： v-??? 开头

数据绑定

Vue中有2中数据绑定方式：

  1. 单向绑定（v-bind），数据只能从data流向页面。
  2. 双向绑定（v-model），数据不仅能从data流向页面，还可以从页面流向data

补充：

  1. 双向绑定一般用在表单类元素上（input、select等）
  2. v-model:value 可以简写为v-model，默认收集的就是value值

el与data的两种写法

el

  1. new Vue的时候配置el属性
  2. 先创建Vue实例，然后通过vm.$mount('#root')指定el的值

data

  1. 对象式 data: { msg: " Hello World " }
  2. 函数式 data(){ return { msg: "Hello World" } }

重要的原则：由Vue管理的函数，一定不要写箭头函数，一旦写了箭头函数，this就不再是Vue实例了。

Vue中的MVVM

MVVM

  • M：模型（model），指的是data中的数据
  • V：视图（view），模板代码
  • VM（viewmodel），Vue实例

观察得：

  • data中的所有属性，都出现在vm身上
  • vm身上的所有属性以及Vue原型上的所有属性，在Vue模板中都可以使用
