    打包完成，可以看到日志上有一段提示：The 'mode' option has not been set

    意思就是，没有配置mode（模式），提醒我们配置

    模式：供mode配置选项，告知webpack使用相应模式的内置优化，默认值为 production ，另外还有 development 、none 他们的区别如下

    选项 描述
    development 开发模式，打包更快，省去代码优化
    production 生产模式，打包较慢，开启tree-shaking和压缩代码
    none 不适用任何默认优化选项
